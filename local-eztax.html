현재 상황:

SSL/HSTS 오류로 인해 직접 접근이 차단되었습니다.

Replit에서 Deploy 버튼을 클릭하여 안전한 도메인을 생성하세요.

접근 방법:

Deploy 방법 (권장)

  1. Replit에서 Deploy 버튼 클릭
  2. 자동으로 안전한 SSL 인증서 생성
  3. 새 도메인으로 접근

임시 접근

Chrome에서:

  1. chrome://flags 접속
  2. "Allow invalid certificates for resources loaded from localhost" 활성화
  3. 브라우저 재시작
